# O-Nakala Core Mobile Guide
## Field Research & Offline Capabilities

### 📱 Mobile-First Features

O-Nakala Core is optimized for field research with comprehensive mobile support:

#### **Progressive Web App (PWA)**
- **Install to Home Screen**: Add O-Nakala to your device's home screen for app-like experience
- **Offline Functionality**: Continue working without internet connection
- **Background Sync**: Automatically sync data when reconnected
- **Push Notifications**: Stay updated on processing status

#### **Touch-Optimized Interface**
- **Swipe Navigation**: Swipe left/right between tabs
- **Pull-to-Refresh**: Pull down from top to refresh data
- **Touch-Friendly Buttons**: Minimum 44px touch targets for accessibility
- **Gesture Support**: Intuitive touch gestures throughout

### 🔄 Offline Capabilities

#### **What Works Offline:**
- ✅ View previously loaded templates
- ✅ Browse cached analytics data
- ✅ Access saved metadata templates
- ✅ View documentation and help content
- ✅ Create new templates (saved locally)
- ✅ Basic file analysis (limited)

#### **What Requires Internet:**
- ❌ Autonomous metadata generation
- ❌ Real-time analytics
- ❌ API-based file uploads
- ❌ Live collaboration features
- ❌ Template synchronization

#### **Offline Data Management:**
- Data is automatically cached for offline access
- Changes made offline sync when reconnected
- Conflict resolution for concurrent edits
- Local storage backup for critical data

### 📋 Field Research Workflow

#### **Step 1: Pre-Field Preparation**
1. **Connect to Wi-Fi** and load O-Nakala Core
2. **Cache Templates**: Visit Templates tab to download available templates
3. **Load Analytics**: Check Analytics tab to cache recent data
4. **Install App**: Add to home screen for offline access

#### **Step 2: Field Data Collection**
1. **Open O-Nakala** from home screen
2. **Upload Files**: Use camera or file picker to select research files
3. **Apply Templates**: Use cached templates for immediate metadata entry
4. **Save Locally**: Data saved automatically to local storage

#### **Step 3: Post-Field Synchronization**
1. **Connect to Internet**: Return to Wi-Fi or cellular coverage
2. **Automatic Sync**: App automatically syncs offline changes
3. **Process Files**: Complete AI-powered metadata generation
4. **Review Results**: Verify and export final metadata

### 🔧 Troubleshooting Mobile Issues

#### **Common Mobile Problems**

**App Won't Install**
- Ensure browser supports PWA installation
- Check available storage space
- Try different browser (Chrome/Safari recommended)

**Offline Mode Not Working**
- Verify service worker registration
- Clear browser cache and reload
- Check browser settings for offline storage

**Slow Performance**
- Close unnecessary browser tabs
- Clear app cache in browser settings
- Restart browser/device
- Check available device memory

**Touch Issues**
- Ensure screen is clean and dry
- Check for screen protector interference
- Verify touch sensitivity settings
- Test in different browser

#### **Performance Tips**
- **Close Background Apps**: Free up memory
- **Use Wi-Fi When Available**: Faster data transfer
- **Regular Cache Clearing**: Prevent storage overflow
- **Update Browser**: Latest features and security

### 📱 Device-Specific Instructions

#### **iPhone/iPad (iOS)**
1. **Open in Safari** (not other browsers for full PWA support)
2. **Add to Home Screen**: Share button → Add to Home Screen
3. **Grant Permissions**: Allow camera/file access when prompted
4. **Enable Notifications**: For sync status updates

#### **Android Devices**
1. **Open in Chrome** (recommended for best experience)
2. **Install Prompt**: Tap "Install" when prompted
3. **Home Screen**: App appears with O-Nakala icon
4. **Permissions**: Allow storage and camera access

### 📚 Field Research Best Practices

3. **Charge Devices**: Ensure sufficient battery life
