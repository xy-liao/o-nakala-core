# o-nakala-core v2.4.3

**First public release**

## What's Included

- **4 CLI tools**: `o-nakala-upload`, `o-nakala-collection`, `o-nakala-curator`, `o-nakala-user-info`
- **Python API**: Complete library for NAKALA repository integration
- **Automated workflows**: CSV-driven batch operations for research data management
- **Quality analysis**: Metadata validation and curation tools

## Installation

```bash
pip install o-nakala-core[cli]
```

## Features

- Upload datasets with metadata to NAKALA
- Create and manage collections
- Batch metadata modifications
- Quality reporting and validation
- Multilingual metadata support (French/English)
- Test environment integration
